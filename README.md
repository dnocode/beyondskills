# BeyondSkills

A personal collection of reusable skills for Claude and AI assistants — archived, versioned, and shared.

## What is a skill?

A skill is a set of instructions you give to an AI at the start of a conversation to change how it thinks and responds. Not a prompt for a single task — a lens that reshapes how the model approaches an entire domain.

Each skill in this repo is:
- **Self-contained** — everything the AI needs is in one file
- **Documented** — a plain-language README explains what it does, when to use it, and how
- **Composable** — skills can be stacked or combined with other instructions

## How to use a skill

1. Browse the [`skills/`](./skills/) directory and find one that fits your need
2. Open the skill's `README.md` to understand what it does
3. Copy the contents of `SKILL.md`
4. Paste it at the beginning of your Claude conversation (or as a system prompt)

That's it. No installation, no tools required.
