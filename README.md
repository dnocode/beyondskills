# BeyondSkills

A personal collection of skills for Claude Code and AI agents — archived, versioned, and shared.

## What are skills?

Skills are reusable prompts/instructions that extend AI agent behavior. They live as `.skill` files and can be invoked during sessions to guide how the agent approaches tasks.

## Skills

| Skill | Description |
|-------|-------------|
| `lateral-thinking.skill` | Lateral thinking techniques for creative problem solving |

## Usage

Skills are designed for use with [Claude Code](https://claude.ai/code) via the Superpowers framework.
